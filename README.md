# 指令伪装（Command Mask）
把“真实指令”映射为“伪装指令”，并可对回复进行静默或自定义替换，适合在对话中伪装成真人。

## 配置面板

- /reset 指令：设置伪装指令、是否静默回复、回复映射。
- /new 指令：设置伪装指令、是否静默回复、回复映射。
- 自定义板块：点击 + 添加自定义规则，填写真实指令、伪装指令、静默回复、回复映射。

说明：
- 伪装指令为空：保持原样，不做映射。
- 关闭静默时，回复映射为空：保持原本输出。
- 只想改回复：把伪装指令填写为与真实指令相同（例如 /reset）。
- 仅在消息被识别为唤醒/指令时生效。

## 示例

- /reset 伪装成 /今天天气真好，静默回复
- /reset 伪装成 /今天天气真好，回复映射为“挺好”

## 兼容旧版

仍支持 `rules_text` 与 `mappings` 旧配置，但已在面板中隐藏。
